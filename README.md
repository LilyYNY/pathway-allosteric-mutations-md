# Pathway Allosteric Mutations and Molecular Dynamics

This repository documents the computational workflow and research ideas for my master's thesis project.

The project focuses on protPheMut-based screening and molecular dynamics validation of distal allosteric missense mutations in key proteins from the RAS/MAPK/PI3K signaling pathways.

## Master's Thesis Project

This repository is associated with my expected master's thesis in Medical Systems Biology at Soochow University.
